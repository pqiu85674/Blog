JavaScript Constructor Function

發表於 2024-08-16 更新於 2024-08-17 分類於 JavaScript

constructor

ES5

  • constructor(建構函數)通常會以大寫字母開頭，這是一種命名慣例，以便與普通函數區分開來
  • this 會指向新創建的物件 你可以在建構函數內部使用 this 來設置物件的屬性
  • 使用 new 關鍵字以創建新的物件實例

要注意的點

  • 不應該把 function 直接寫在函式建構式當中，否則每個被 new 出來的物件都包含此 function，這樣非常浪費記憶體空間
  • Person.prototype 上定義的方法可以被所有 new 出來的物件所共享，這樣可以節省記憶體空間

ES6

轉換成 ES6

這裡 greet function 雖然沒有使用 prototype，但是 ES6 的 class 會自動添加到 prototype 上
也就是只存在一個 function，可供其他物件呼叫
console.log(Alice.greet === Bob.greet);//true 指向一樣的位址

封裝

ES5

注意

  • 使用 const 宣告變數，不使用 this， 變數名稱前綴要加上_表示 private
    並且讓內部 function 呼叫目的就是為了讓外部取不到資料

  • function 故意不使用 prototype，使用 this.getInfo 是為了要取得內部資料
    this._name and this._age
    若使用 prototype 會取不到 this._name and this._age

ES6

注意

  • 這裡也是宣告變數，並且使用_表示 private
  • 注意this.getInfo = () => console.log(_name, _age);
    是放在 constructor 內部，不是放在 constructor 外部
    放在 constructor 外部是使用 prototype，也就是可供所有實例呼叫的 function
    但是放在 constructor 外部是呼叫不到內部的this._name and this._age
    所以要放在 constructor 內部

繼承

ES5

講解

  • 先宣告 Dog function 使用 const _變數名稱 並使用 this.function 來取得內部資料 來實現封裝

  • Dog.prototype.run把 function 建立在 prototype 上，使其不會跟隨 new 出來的物件
    來達到記憶體優化效果

  • 再宣告 DogChild function 這裡參數要放入想父層的參數，也就是 name 和 color，加上本身的參數 age

  • Dog.call(this,name,color);這裡是放父層的資訊

  • DogChild.prototype = Object.create(Dog.prototype);
    這行是讓 DogChild 繼承 Dog 的屬性及方法
    其中Object.create(Dog.prototype);會建立一個新的對象
    這個對象的原形是 Dog.prototype，所以此對象會繼承 Dog 的方法及屬性
    而DogChild.prototype被賦予新的對象，這樣 DogChild 的所有實例就能夠訪問 Dog 原型上的方法

  • DogChild.prototype.constructor = DogChild;
    使用DogChild.prototype = Object.create(Dog.prototype);的時候
    DogChild.prototype.constructor被設為 Dog
    因為 DogChild.prototype 的原型已經被改變了
    這代表 DogChild 實例的 constructor 屬性會指向 Dog 而不是 DogChild

    這行程式的作用是手動修正 DogChild.prototype.constructor 為 DogChild
    這樣 DogChild 的實例就可以引用它們的構造函數 DogChild

  • 後續宣告const dogChild = new DogChild("dog", "black", 18);
    dogChild 就可繼承 Dog 方法及物件

在 class 後面接 extends 上層 class
並且在 constructor 裡面使用 super 來引入上層 class 參數
